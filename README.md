- email address : sksowk156@gmail.com <br />

## Features
WatchOut은 졸음을 방지 서비스입니다.<br />
졸음을 감지하면 경고음을 울리고, 주변 휴식 공간 정보를 제공합니다.<br />
경고음을 커스텀할 수 있고, 졸음 감지 결과를 확인할 수 있습니다.<br />
